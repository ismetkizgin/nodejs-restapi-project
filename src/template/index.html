Get started

                    API Domain

Solver API provides access to solver algorithms.

Installation Instructions


#package install
$ npm install

#project build
$ npm build

Solver Api, install and compile project packages

Countdown Solver


# Request Example
GET countdownsolver/[2,4,5,7,5,50]/425

For the countdown solver, you need to make a GET call to the URL below :
countdownsolver/[2,4,5,7,5,50]/425 



Result example:
{
    "result": "50 * 4 * 2 = 400\n5 * 5 = 25\n400 + 25 = 425\n",
    "score": 9
}

Word Solver


# Request Example
GET wordsolver/[m, e, k, t, u, a,g, y]

For the word solver, you need to make a GET call to the URL below :
wordsolver/[m, e, k, t, u, a,g, y] 



Result example:
{
    "word": "MUKTEZA",
    "score": 9
}
